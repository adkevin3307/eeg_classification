# EEG Classification

EEG classification w/ PyTorch

## Environment

- `python >= 3.8`
- `torch >= 1.8.0`
- `torchvision` w/ appropriate version

## Usage

- `python main.py [-n|--net] {net architecture, EEG|DeepConv, default=EEG} [-a|--activation] {activation function, ReLU|LeakyReLU|ELU, default=ReLU}`
